以下是我今天在安裝 Gnome 和 Chrome 時，在 chatgpt 查到的資訊，我也是按照這些指令作

1. 安裝 Gnome-Desktop 比較簡單，只有一個指令

   sudo apt install ubuntu-desktop

2. 使用 ssh 連結 remote ubuntu 的 Gnome 需要修改參數

   2.1 On the Remote Ubuntu Machine:

       2.1.1 Install OpenSSH Server (if not already installed):
             sudo apt update
             sudo apt install openssh-server

	   2.1.2 Enable X11 Forwarding:
             sudo nano /etc/ssh/sshd_config

             Inside the file, make sure you have the following lines:
             
             X11Forwarding yes
             X11DisplayOffset 10

       2.1.3 Restart SSH Service:
             sudo service ssh restart
	   
   2.2 On Your Local Machine (Linux):
       
	   2.2.1 Open an SSH Connection with X11 Forwarding: (以 node1 為例)
	         ssh -p 5021 -X ntut01@59.124.122.57
			 
	   2.2.2 Start GNOME:
	         gnome-session
			 
3. Install Chrome:
   3.1 Open a Terminal: 
   
   3.2 Update Package Lists: 
       sudo apt update

   3.3 Install Dependencies (optional):
       sudo apt install libxss1 libappindicator1 libindicator7

   3.4 Download Google Chrome:
       wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb

   3.5 Install Google Chrome:
       sudo dpkg -i google-chrome-stable_current_amd64.deb
