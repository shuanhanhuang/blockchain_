1. 安裝 Ubuntu Server
  - 前往 Ubuntu 官方網站下載Ubuntu：https://ubuntu.com/download/server, 下載適合的 Ubuntu Server 安裝檔
  - 準備一個空的 USB 隨身碟, 將 Ubuntu Server 映像檔寫入 USB 隨身碟中
  - 在電腦的 BIOS 設定中將開機順序設定為先讀取 USB
  - 插入 Ubuntu 開機 USB，開機並按照提示進行安裝，將電腦安裝成 Ubuntu 作業系統的電腦
  - https://github.com/Jenniferwyqq/BlockChain/blob/main/ubuntu_server.jpg

2. 安裝 Ubuntu Desktop
  - 更新系统
    >>> sudo apt update
    >>> sudo apt upgrade
  - 安装GNOME桌面环境
    >>> sudo apt install ubuntu-gnome-desktop
  - 在安装过程中，会提示选择显示管理器。当被询问时，选择"gdm3"选项。
  - 安装完成后，重新启动系统
    >>> sudo reboot
  - 重啟後，會進入登錄界面。 單擊登錄框旁邊的齒輪圖標，然後選擇“GNOME”作為桌面環境。
  - 使用您的用戶名和密碼登錄，GNOME X11 將加載。

  
3. 安裝 Chrome
  - 開啟終端機 (Terminal)
  - 切換至下載目錄
    >>> cd Downloads
  - 下載 Google Chrome 安裝檔案
    >>> wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
  - 安裝 Google Chrome
    >>> sudo dpkg -i google-chrome-stable_current_amd64.deb
  
4. 檢查連線到的 Wi-Fi 存取點 (AP)
  - 開啟終端機 (Terminal)
  - 執行以下指令檢查 Wi-Fi AP
    >>> sudo iwlist wlo1 scan grep ESSID
  - 使用 iwconfig 指令檢查目前的 Wi-Fi 連線狀態
    >>> iwconfig

以上是安裝 Ubuntu Server 和 Ubuntu Desktop 和一些常用指令的基本介紹。如果需要更詳細的資訊，可以參考以下連結提供的文章：https://ithelp.ithome.com.tw/articles/10192420 或 https://ubuntu.com/tutorials/install-ubuntu-desktop#1-overview
